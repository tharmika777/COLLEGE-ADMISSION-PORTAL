# 🎓 College Portal (HTML, CSS, JavaScript)

A simple **College Portal Web Application** built using **HTML, CSS, and mainly JavaScript**.  
This project includes **Student and Admin login**, a dashboard with multiple sections, and **LocalStorage support** to store marks and notices without using any database.

---

## 🚀 Features

### ✅ Student Features
- Login as Student
- View Student Profile
- View Attendance Percentage
- View Marks (Subject-wise)
- View College Notices
- Logout option

### ✅ Admin Features
- Login as Admin
- View all student dashboard sections
- Add new notices
- Update marks (add new subjects or edit existing)
- Changes are saved using **LocalStorage**

---

## 🛠 Technologies Used

- **HTML5** – Page structure
- **CSS3** – Styling and layout
- **JavaScript (ES6)** – Functionality and dynamic content
- **LocalStorage** – Data storage (Marks & Notices)
